# OKX Hybrid Trading Bot

Automated trading bot for OKX using a hybrid scoring strategy (RSI, Z-score, EMA, volume analysis).

## Features
- Paper trading mode
- Trades BTC, ETH, SOL
- CSV trade logging
- Dockerized for easy deployment

## Deployment on Coolify
1. Create new resource → Git Repository
2. Add environment variables
3. Deploy
